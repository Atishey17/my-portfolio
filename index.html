My Projects

AI Assistant

This Python script creates a voice assistant web application using Streamlit. Users can select a language and type questions into an input box. The input is sent to OpenAI's GPT-3.5 model to generate a response. The response is displayed as text and played back as speech using text-to-speech. Additional features include a button to open a browser window for typing input.

Encoder-Decoder

This script implements a simple encryption and decryption tool using Tkinter. Users input text and a secret key, which are then encrypted or decrypted. Encryption converts the text into a base64-encoded string. Decryption reverses the process, decoding the base64 string back into text. The interface allows users to reset input fields and perform encryption or decryption actions.

URL Shortner

The project initializes an Express application and configures middleware for request handling. Routes for the root and users are defined, along with view engine setup using ejs. Middleware functions for logging, parsing JSON and URL-encoded data, and handling cookies are applied. Static file serving is set up to serve files from the public directory.

ASCII Image

This fun python code utilizes OpenCV to capture video from the webcam. Each frame is converted to grayscale for ASCII art conversion. ASCII art conversion is performed by mapping pixel values to characters. The converted ASCII art is displayed in a Tkinter window. The program continuously updates the ASCII art display.
